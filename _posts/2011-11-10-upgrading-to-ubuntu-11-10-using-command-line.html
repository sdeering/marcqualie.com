--- layout: post title: "Upgrading to Ubuntu 11.10 using Command Line" date: 2011-11-10 21:23:00 tags: --- Here is a very simple script to update from Ubuntu 11.04 to 11.10. I have just launched a few new servers and this was the first thing I had to do on all of them, so I thought I'd share.
apt-get update
apt-get autoclean
do-release-upgrade

Some of the above code might not be needed on a fresh install, but I always like to make sure the packages are all up to date before upgrading to a new distribution. This will work on both Desktop and Server editions of Ubuntu 11. Also, remember to use sudo if you're not logged in as root.
