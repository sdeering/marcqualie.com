To start off with, my slider class has been very popular so that's what I'm going to share first. This class is in use on about 20 domains, so it's rather stable. Below is the simple code to get a slider running straight away.

var MySlider = new SliderClass({
slideDiv: 'featured',
slideClass: 'f-pic',
leftArrow: '.arrow-left',
rightArrow: '.arrow-right',
useKeyboard: true,
autoSlide: true,
loopBack: true
});

That is Javascript, obviously you can change the values to customize it to your project. Now onto the HTML.






You can pretty much do what you like with the CSS, but what ever you specify as SlideClass must be position absolute, and SlideDiv must be position: relative. These elements must also be the same width and height, otherwise the Slider won't line up properly.
