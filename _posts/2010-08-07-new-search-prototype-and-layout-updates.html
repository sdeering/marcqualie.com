Now for the main feature, my brand new Search Engine. Not only does this index the content on Marc Qualie, and my Blog, but it can index the entire web. I basically coded a bot to crawl pages and index them acording to a system such as link algorithms, meta data, and open graph information. This is then compressed, stored, then indexed by a second robot to get instant results for any combination of key words. You can test out it's usefulness here. As you can see Adobe AIR posts come first, then the Adobe label, then an air based game, followed by an android tag. I never said it was perfect, but this is still an early prototype, and very relavent results are displayed in microseconds (that's fast!).

You may also notice the game results, and the homepage have thumbnails to show more interactive content. This is another algorithm I have been working on, taking data from open graph info, and media content within Blog etc. This is then checked against incoming link titles, and alt tags on images, to display the most relevant thumbnail. Clever stuff, but I still have so much more planned. If you have any suggestions please let me know as I am hoping to release this engine on a large scale when I get better web hosting.
